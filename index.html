AcessibiWeb - Diretrizes de Acessibilidade na Web

A WEB é uma poderosa ferramenta de inclusão, por este motivo existem diretrizes de acessibilidade web, que podem ser definidas como recomendações que possuem o objetivo de tornar a construção de Web Sites e Conteúdos na Web mais acessíveis a todos os usuários. Esta página reune diversas diretrizes de acessibilidade web categorizadas de acordo com as deficiências as quais podem oferecer suporte quando utilizadas em sites e conteúdos.
